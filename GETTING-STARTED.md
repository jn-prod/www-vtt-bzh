# Getting started

## Clone repo

## Installation

```sh
git clone https://github.com/jn-prod/www-vtt-bzh.git
corepack enable
pnpm i
```

## Tests

```sh
pnpm test
```

## Release

### Create new changeset

```sh
pnpm change
```

### Preview release

```sh
pnpm release:preview
```

### Release

```sh
pnpm release
```
